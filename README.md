le Snake Mod
====
A modified fun-infused version of the classic **Snake** game. There's food, there's snake and there's more!
Avoid the falling blockades and eat your way to glory!

---

####Compilation
Run
```bash
make
```
to compile and
```bash
./le_snake_mod
```
to run.
